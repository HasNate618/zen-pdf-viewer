# Zen PDF viewer (Omarchy theme)

Location: `~/.local/share/pdfjs` (symlinked to this project)

Usage: `zen-pdf-viewer /path/to/file.pdf`

Components:
- `viewer.html` (PDF.js-based Omarchy viewer)
- `~/.local/bin/zen-pdf-viewer` (launcher script)

Purpose: local PDF.js viewer with Omarchy-styled Zathura-like mode (grayscale + invert, transparent page background) and selectable text.

To share: push this repo to a remote (see instructions below) or copy the directory.
